# AXA Assignment Frontend

## Installation

```bash
$ npm install
```

## Before running

Create a `.env` file in the root directory and add the following environment variables:

| Key           | Example Value                          |
| ------------- | -------------------------------------- |
| VITE_BASE_URL | https://axa-assignment-bff.vercel.app/ |

## Running the app

```bash
$ npm run dev
```

## Test

```bash
$ npm run test
```
